## 📘 Descripción

Este repositorio contiene el Backend General del Sistema de la Gerencia de Desarrollo Humano (SGDH), desarrollado por el equipo de desarrolladores de la Central de Control y Monitoreo de la Municipalidad de San Juan de Lurigancho.

El sistema está construido utilizando el framework  [NestJS](https://github.com/nestjs/nest), bajo una arquitectura modular que facilita el mantenimiento y escalabilidad del proyecto.

Permite gestionar las distintas áreas de la Gerencia de Desarrollo Humano en base a los roles de los usuarios asociados.

---
## 🔑 Características principales

- Registro y gestión 
- Asignación de roles
- Visualización de personal
- Filtros destacados

---

## 🔑 Configuración del Proyecto

```bash
$ pnpm install

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
